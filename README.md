一个数据驱动的`UICollectionView`框架，用于构建快速灵活的列表。

|         | 主要特性  |
----------|-----------------
&#127968; | 更好的可复用 cell 和组件体系结构
&#128288; | 创建具有多个数据类型的列表
&#128241; | 简化并维持`UICollectionView`的核心特性
&#9989;   | 超多的插件来帮助你构建更好的列表
&#128038; | Swift 编写, 同时完全支持 SwiftUI
